Software Engineering

The software coordinates user inputs, hardware control, and inventory tracking:

  • Keypad Logic: Interprets user input, debounces entries, locks during dispensing.
  • Motor Control: Triggers motor rotation; includes retry loop via IR feedback.
  • Sensor Integration: Detects item drops; unlocks input or retries if dispensing fails.
  • Inventory Tracking: Logs item counts and triggers LED alerts on low stock.
  • Portfolio Website: Hosted on GitHub Pages, showcasing our full engineering journey.
