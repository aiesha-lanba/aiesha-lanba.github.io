Software Engineering

Frontend

The frontend refers to the user-facing interface of the vending machine, primarily managed through the membrane keypad and LED feedback system.

  • Keypad Input: Users interact with a 1x4 membrane keypad to select items. The interface ensures single keypress detection with a debounce mechanism and locks input during dispensing to avoid conflicts.
  • LED Indicators: LED strips provide visual feedback—highlighting the machine's operational status and indicating low inventory by changing color when stock drops below threshold.
  • User Feedback: Keypad responses, visual cues from LEDs, and retry prompts collectively ensure intuitive user interaction.

Backend

The backend includes all logic executed on the Arduino microcontroller to control the vending machine’s behavior.

  • Motor Control Logic: When a selection is made, the corresponding servo rotates to dispense the item. The system uses an IR sensor to confirm success and retries automatically if no item is detected.
  • Sensor Handling: The IR sensor continuously monitors the retrieval zone. If a dispense attempt fails twice, the system unlocks the keypad and allows a new selection.
  • Inventory Management: The system maintains a local hashmap tracking each item’s remaining quantity. This data controls LED color changes and prevents out-of-stock items from being selected.
  • System Stability: Includes lockout protection to prevent simultaneous operations, automatic retry cycles, and a reset protocol post-dispense to prepare for the next transaction.
