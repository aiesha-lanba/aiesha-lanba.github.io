Vendi

Our team built a custom vending machine that addresses the lack of accessible, healthy snack options for students on campus. This interdisciplinary project integrates mechanical engineering, electrical systems, and software design, providing hands-on experience in real-world systems integration.

The vending machine will:

  • Use servo-controlled spiral spring dispensers
  • Be enclosed in a modular polycarbonate frame
  • Include an IR sensor to detect successful dispensing
  • Feature a keypad-based user interface and onboard data tracking
  • Be documented and showcased through this website
